# 📱 Dody Store – Flutter E-commerce App

**Dody Store** is a modern and scalable e-commerce mobile application built using Flutter.  

The app provides a smooth and intuitive shopping experience, allowing users to explore products, manage their cart, and complete purchases easily and professionally.

---

## 🧠 Architecture

It is designed following the **MVVM architecture pattern** with **GetX** for efficient state management and dependency injection, ensuring clean, maintainable, and high-performance code.

---

## 🚀 Features

- 🔐 **User Authentication**
  - Secure login & signup with form validation  

- 💾 **Caching Strategy (sqflite)**
  - Local data persistence for cart & preferences  

- 🎨 **UI/UX**
  - Clean, responsive, and modern design  

- 📦 **Order Management**
  - Order history tracking  

- 💳 **Checkout System**
  - Complete purchase flow  

---

## 🛠️ Tech Stack

- Flutter & Dart  
- GetX (State Management & Dependency Injection)  
- sqflite (Local Database for cart & user preferences)
